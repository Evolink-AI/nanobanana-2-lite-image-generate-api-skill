# Code of Conduct

## Our Standard

This repository should stay useful, focused, and safe for developers integrating EvoLink APIs and agent skills.

Participants are expected to:

- Be respectful and specific when reporting problems.
- Keep discussions focused on EvoLink API integration, examples, packaging, and agent skill behavior.
- Avoid posting API keys, private task outputs, generated private URLs, or personal data.
- Provide reproducible evidence when reporting bugs.

Unacceptable behavior includes harassment, abusive language, spam, secret exposure, and attempts to misuse the installer or API examples.

## Enforcement

Maintainers may edit, hide, or remove comments and issues that expose secrets, contain abuse, or are unrelated to this repository's scope.

For security-sensitive reports, use the process in `SECURITY.md` instead of opening a public issue.
